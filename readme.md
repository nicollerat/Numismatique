# Comment jouer ?

Le script graphique.ipynb de type jupyter lab. Il peut être visionné sur github.

Pour l'éditer, il faut installer python et quelques librairies. Lorsqu'il est éxécuté, il va lire la feuille xlsx. Si la feuille est modifiée, le script ne se met pas tout seul à jour sur github.

"Positron" est un éditeur pour ce genre de fichier avec des plug in "jupyter lab"
